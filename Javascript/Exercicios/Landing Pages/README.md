# Exercícios de Front End

### Exercícios de HTML e CSS feitos em sala de aula no SENAI.

O exercício é composto por uma página principal (index), com âncora para 3 outras páginas:
* Fasfood - Landing Page para uma rede de fast food.
* Cardápio - Pequena página de um cardápio onde consta imagem, descrição e preço.
* Perfil - perfil pessoal com links de direcionamento a páginas externas.

#### Alguns conceitos abordados:
* Tipografia
* Paleta de cores
* Imagens
* Estilizar tags separadamente
* Alinhamento com flex
